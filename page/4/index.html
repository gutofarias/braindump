The cut-set forms a matrix where each column relates to an edge of the Action Graph, so there are as many columns as edges. For each row we cut one branch and assess the positiveness of the cutted elements. Each column in the row is either +1 if the respective element was cutted positively, -1 if cutted negativelly and 0 if not cutted. There is one row for each branch in the Coupling Graph.
